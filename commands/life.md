Start a life planning session (adhoc, not daily).

## Steps

1. **Read context files (if they exist)**
   - `life/travel.md`
   - `life/ideas.md`
   - `life/holiday-planning.md`
   - Check Things 3 inbox and todos (if Things 3 MCP is available)

2. **Review what's there**
   - Summarize upcoming travel/events
   - List any pending ideas/projects
   - Show Things 3 reminders if readable

3. **Collect new items**
   - Ask: "Any new reminders, travel plans, ideas, or things to track?"
   - For time-sensitive items → create in Things 3 with reminder time (if available)
   - For reference items → add to appropriate markdown file

4. **Update files**
   - Add new items to relevant files (creating files if they don't exist)
   - Clean up completed items if mentioned

## Things 3 Usage (if available)
- Use `when` parameter with time (e.g., "2025-12-10 09:00") for actual notifications
- Creation works; reading may be unreliable

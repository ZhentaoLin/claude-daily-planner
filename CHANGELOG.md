# Changelog

All notable changes to the Daily Planner plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-07

### Added
- Initial release of Daily Planner plugin
- `/plan` command with adaptive behavior (full planning vs casual check-in)
- `/checkin` command for quick task suggestions
- `/strategy` command for big picture planning
- `/show` command to display today's plan
- `/review` command for end-of-day reviews
- `/idea` command for capturing side project ideas
- `/life` command for personal life planning
- Setup script for initializing planner directory structure
- Comprehensive documentation and README
- Calendar integration support via icalBuddy (optional)
- Procrastination detection and intervention strategies
- Recurring deferral flagging (tasks deferred 3+ days)
- Weekly review mode (Monday mornings)
- Flexible file structure support

### Philosophy
- Planning that adapts to your energy and context
- Reduces friction instead of adding structure
- Context-aware and pattern-recognizing
- Procrastination-friendly with practical strategies
